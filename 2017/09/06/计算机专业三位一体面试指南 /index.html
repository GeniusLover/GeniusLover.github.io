计算机专业三位一体面试指南

电气通信与计算机类

  • 电气工程及其自动化、自动化、电子信息工程、通信工程、电子科学与技术
  • 计算机科学与技术、软件工程、网络工程、数字媒体技术、物联网工程

目前流传的王道专业：计算机科学与技术、软件工程
其中计算机科学与技术又优于软件工程。除了那两个，余下的都是计算机相关专业。

专业介绍

简单的先总结一下。计算机科学和软件工程，对口职业是程序员；通信工程、电子信息工程、电子科学与技术，对口职业是华为、中兴、移动等通信企业，德州仪器、因特尔、清华紫光等芯片制作商，但一部分也去做了程序员；网络工程、物联网工程估计是新专业，顾名思义是去做网络、嵌入式、单片机方面的研究，数字媒体技术，做多媒体渲染方面研究，对口职业是 动画制作、游戏制作等；
以上都是弱电相关专业，电子工程及其自动化、自动化是强电相关，对口单位有国家电网、电机制作等等。当然什么行业都可以转行去做程序员。

从目前的形式来看，专业选择顺序：计算机科学>软件工程>网络工程=数字媒体技术=通信工程=电子科学与技术=物联网工程=电子信息工程>强电相关专业。

未来的话，国家正在扶持集成电路产业、5G正在成熟中，我也说不好未来走势，不过计算机科学与技术在可预见的未来都是不错的。

  • 计算机科学（英语：computer science，有时缩写为CS）是系统性研究信息与计算的理论基础以及它们在计算机系统中如何实现与应用的实用技术的学科。[1][2]它通常被形容为对那些创造、描述以及转换信息的算法处理的系统研究。计算机科学包含很多分支领域；有些强调特定结果的计算，比如计算机图形学；而有些是探讨计算问题的性质，比如计算复杂性理论；还有一些领域专注于怎样实现计算，比如编程语言理论是研究描述计算的方法，而程序设计是应用特定的编程语言解决特定的计算问题，人机交互则是专注于怎样使计算机和计算变得有用、好用，以及随时随地为人所用。
    有时公众会误以为计算机科学就是解决计算机问题的事业（比如信息技术），或者只是与使用计算机的经验有关，如玩游戏、上网或者文字处理。其实计算机科学所关注的，不仅仅是去理解实现类似游戏、浏览器这些软件的程序的性质，更要通过现有的知识创造新的程序或者改进已有的程序。
  • 软件工程（英语：Software Engineering[1]）1968年秋季，NATO（北约）的科技委员会召集了近50名一流的编程人员、计算机科学家和工业界巨头，讨论和制定摆脱“软件危机”的对策。在那次会议上第一次提出了软件工程（software engineering）这个概念，研究和应用如何以系统性的、规范化的、可定量的过程化方法去开发和维护软件，以及如何把经过时间考验而证明正确的管理技术和当前能够得到的最好的技术方法结合起来的学科。它涉及到程序设计语言、数据库、软件开发工具、系统平台、标准、设计模式等方面。
    在现代社会中，软件应用于多个方面。典型的软件比如有电子邮件、嵌入式系统、人机界面、办公包、操作系统、编译器、数据库、游戏等。同时，各个行业几乎都有计算机软件的应用，比如工业、农业、银行、航空、政府部门等。这些应用促进了经济和社会的发展，提高人们的工作效率，同时提升了生活质量。
    软件工程师是对应用软件创造软件的人们的统称，软件工程师按照所处的领域不同可以分为系统分析师、系统架构师、软件设计师、程序员、测试工程师、界面与交互设计师等等。人们也常常用程序员来泛指各种软件工程师。
  • 通信工程（也作信息工程、电信工程，旧称远距离通信工程、弱电工程）是电子工程的重要分支，同时也是其中一个基础学科。该学科关注的是通信过程中的信息传输和信号处理的原理和应用。
    通信工程研究的是，以电磁波、声波或光波的形式把信息通过电脉冲，从发送端（信源）传输到一个或多个接受（信宿）。接受端能否正确辨认信息，取决于传输中的损耗高低。信号处理是通信工程中一个重要环节，其包括过滤，编码和解码等。
    通信工程所关注的频段涉及甚广。低频段，亦即低赫兹，关心的是技术声学或低频技术。高频段中关注的范围从微波或雷达系统到可见光的激光或镭射系统。微波到可见光中间的频段几乎都是通信工程的研究对象。除此之外，通信过程中所应用的媒介和技术，包括通信系统在陆上、水下、空中和宇宙空间中的应用，也是相当丰富的。
    通信工程的基础建立于应用数学中的数理方程。其理论起点是物质与波在傅里叶热扩散和麦克斯韦电动力条件下观察到的传播现象。
    世界上由人类创造的最大的通信系统是公共交换电话网（PSTN）。另外一个正在迅速发展的大型通信系统——英特网——正逐步形成电话网的规模，并终将有一天取而代之。
    不可否认的是，如今通信工程正在转变为信息工程，英特网就是一个很好的例子。一方面通信系统常常是信息与应用技术的计算系统的一个重要组成部分，另一方面现代通信系统给信息工程予以理论和方法指导，并处于计算系统的核心位置。
  • 网络工程是设计和实施计算机网络的信息技术专业人员。一般而言，网络工程师不进行日常的的系统管理工作，而是专注于网络设备和基础设施，例如路由器、网络交换机、线缆、介质和网络设备连接用的的协议。 网络工程师的职责还包括，网络的规划，组建，设计，网络设备的安装调试，以及后期的网络故障排除，及网络的维护。

计算机学院选拔要点：

  1. 数学与英语要好
  2. 逻辑思维要强（回答问题时要体现这一点，全面不遗漏）
  3. 对计算机编程方面有兴趣爱好为佳，有天赋最好 （这个的话一般他会问你的兴趣爱好有哪些？之前做过哪些和计算机有关的业余研究？）
  4. 主动学习能力、自学能力

可能面试时会遇到的问题：

  • 在计算机领域有你的偶像吗？为什么他是你的偶像？

  • 你可以用英语自我介绍一下吗？

  • 你对计算机领域现在的前沿研究有了解吗？

    人工智能、数据挖掘、神经网络、大数据分析，etc

  • 给你一个逻辑问题要求你解答？

    题目类型可能像这样：三个嫌疑犯在法官面前各执一词，甲说：乙在说谎；乙说：丙在说谎；丙说：甲乙都在说谎。法官为了难，到底谁在说谎？

  • 你平时有哪些和计算机相关的兴趣爱好呢？

    是否参加过计算机竞赛；是否自学过编程语言（哪怕最后感觉就只写过hello world也可以）；是否自己主动关注过计算机领域的相关新闻；是否对拆解电脑主机感兴趣；是否尝试过重装电脑系统；是否尝试过翻墙去使用google；如果自学过Photoshop、视频制作也可以，etc

  • 你认为人工智能可能会在哪些方面改变人们的生活？

  • 你对自己未来的规划是怎么样的？

    老师希望听到你是真的想要好好的研究计算机科学。最好你想过大学做什么，毕业后去做什么，最后在计算机专业领域达成什么样的成就（学术界、工业界）

  • 如果我们给了你进入计算机学院的机会，你会怎么规划你的大学生活？

    好好学习专业课，因为计算机学院的专业课和生产实践结合的很紧密，打好基础才能走的更远；我会参加一些黑客马拉松、阿里巴巴天池数据分析大赛、ACM等等编程比赛；我会去相关互联网企业实习，在实践中提高自己的团队合作能力、编程能力和工程能力

  • 你认为你进入计算机学院的优势是什么？

    1. 因为热爱（一定要给自己做过哪些事情作为例证）
    2. 因为计算机领域有我的偶像，我想要像他那样……
    3. 因为计算机行业是我认为最容易改变世界的行业（举几个例子。表现一下自己的信心和野心也是个好事，如果面试老师不是计算机学院老师的话，考虑下要不要说这点）

  • 你的性格？

    合作、细心、勇于挑战、喜欢创新，etc
